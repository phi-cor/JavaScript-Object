# Kanban Board

## Install

Download the repository

```bash
git clone https://github.com/seeren-training/JavaScript-Object
```

Change the directory

```bash
cd kanban-board
```
Install packages

```bash
npm install
```

## Run

Start the server

```bash
npm start
```
